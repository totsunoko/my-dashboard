## 変更内容

### 1. 全体テーマの刷新
- **カラーパレット**: ダークブラウン (`#1a1614`) を基調とした落ち着いたトーンに変更。
- **レイアウト**: 3カラム・2行のグリッドシステムを採用し、画像通りのコンポーネント配置を実現。
- **カードデザイン**: 角丸の半径を大きくし、立体感のあるシャドウを適用。

### 2. 時計カードのボーダーアニメーション
- カードの縁を白い光がなぞるアニメーションをCSSの `conic-gradient` を使用して実装。
- ダイナミックかつミニマルな表現にしました。

### 3. 各コンポーネントの高品質化
- **天気**: 3時間ごとの予報を含む詳細なレイアウト。
- **鉄道・ニュース**: リスト形式で整理し、読みやすさを向上。
- **カレンダー**: 現在の日付を白いボックスで際立たせる画像通りのスタイル。
- **音楽プレーヤー**: アルバムアート、曲名、アーティスト名、歌詞、シークバー、操作ボタンを美しく配置。

### 4. スリープ防止機能 (Screen Wake Lock)
- **自動スリープの抑制**: Screen Wake Lock API を実装し、ダッシュボードを表示している間はタブレットが自動的にスリープしないようにしました。
- **堅牢な制御**: 他のタブに切り替えてから戻った際や、長時間放置した後でも、ページが前面にある限り有効性を維持します。

### 5. コードの構造化とリファクタリング
- **ファイルの外部化**: 肥大化していた `index.html` を、役割に応じて `css/style.css` と `js/app.js` に分割しました。
- **可読性の向上**: 各言語が独立したファイルになったことで、将来的な機能拡張やメンテナンスが容易になりました。

### 6. 焼付き防止機能 (Screen Burn-in Prevention)
- **ピクセルシフト**: 1分ごとに画面全体を最大3ピクセル程度ランダムに移動させることで、同じピクセルが同じ色を発光し続けるのを防ぎます。
- **輝度ブリージング**: 10分周期で背景の明るさを極めてゆっくりと変化させ、パネルへの負荷を緩和します。目には見えないレベルの微調整で視認性を損ないません。

### 7. リアルタイム天気予報 (新宿)
- **地点の変更**: 予報地点を大阪から「新宿」に変更しました。
- **ライブデータの統合**: Open-Meteo API と連携し、現在の気温・天気に加え、3時間ごとの予報をリアルタイムに取得・表示するようにしました。
- **自動更新**: 1時間ごとに最新の気象情報を自動取得します。

### 8. 天気予報アイコンのモノクロ化
- **デザインの統一**: カラーの絵文字を避け、CSSフィルタ（グレースケールと輝度調整）を適用することで、すべての天気アイコンを白一色のトーンに統一しました。
- **視覚的プレミアム感**: ダークテーマの背景に馴染む、落ち着いたモダンな外観を実現しました。

### 9. ニュースの外部データ連携
- **ライブヘッドライン**: Google ニュースのRSSフィードをリアルタイムに取得し、最新の主要ニュースを表示するようにしました。
- **レイアウトの最適化**: 鉄道セクションを削除し、ニュースリストに割くスペースを広げることで、より多くの見出しを視認できるようにしました。
- **動的更新**: 30分おきに自動でニュースを更新します。
